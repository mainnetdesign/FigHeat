# FigHeat – Publishing Assets

Assets para publicação na Figma Community.

## Arquivos incluídos

| Arquivo | Uso |
|---------|-----|
| `icon-128.png` | Ícone do plugin (128×128) |
| `thumbnail-1920x1080.png` | Thumbnail principal da página do plugin |
| `screenshots/` | Adicione 3–9 capturas de tela do plugin em uso |

## Como usar

1. No Figma Desktop: **Plugins** → **Development** → **Publish** (ou equivalente)
2. No modal de publicação, faça upload de:
   - **Ícone:** `icon-128.png`
   - **Thumbnail:** `thumbnail-1920x1080.png`
   - **Screenshots:** tire capturas do plugin em ação e adicione aqui

## Dica para screenshots

Execute o plugin no Figma, tire screenshots de:
- Tela de upload
- Heatmap sobre uma landing page
- Modo A/B
- Export para o canvas
